# 🃏 Poker Organizer

Fullstack webová aplikace určená pro kompletní správu a automatizaci pokerových turnajů. Projekt vznikl jako řešení reálné potřeby nahradit neefektivní papírové záznamy digitálním nástrojem.

## 🚀 Hlavní funkce
- **Správa blindů:** Automatické hlídání struktury a navyšování sázek.
- **Timer & Blind Structure:** Přehledný časovač pro hráče.
- **Persistence stavu:** Díky knihovně **Zustand** (s persist middleware) aplikace neztratí data ani po obnovení stránky.
- **Moderní UI:** Plně responzivní rozhraní vytvořené v **Tailwind CSS**.

## 🛠️ Použité technologie
- **Framework:** Next.js (App Router)
- **Jazyk:** TypeScript (96.3% projektu)
- **State Management:** Zustand
- **Styling:** Tailwind CSS

## 📖 Moje cesta k projektu
Tento projekt pro mě byl klíčový v pochopení **State Managementu** v Reactu. Musel jsem vyřešit, jak udržet synchronizovaný časovač a stav hry napříč celou aplikací, aniž by došlo ke ztrátě dat při interakci uživatele. 

## 🏁 Instalace a spuštění
1. `npm install`
2. `npm run dev`
